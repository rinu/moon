# moon

## Project setup
```
yarn install
```

### Configuration
```
cp .env .env.local
```
Get an auction id from osta.ee and set it in `.env.local`

The auction id can be overridden later with the `auction` URL parameter.

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
